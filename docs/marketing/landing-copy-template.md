# Plantilla de Contenidos para Landings Premium (Marketing)

Este formato es para que el equipo de marketing llene textos de una nueva landing sin tocar estructura.

Regla de palabras:
- Cada campo tiene un OBJETIVO de palabras.
- Se permite un rango de OBJETIVO -10 y OBJETIVO +10.
- Si el objetivo es menor a 10 palabras, usar criterio natural (frase corta y clara).

No editar en esta plantilla:
- CTAs (botones): se quedan como ya estan en el sitio.
- Rutas, links internos, IDs, nombres tecnicos, imagenes, y estructura.
